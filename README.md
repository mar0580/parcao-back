
# 🐾 Parcão Back

## 📋 Sobre
O Parcão é uma API REST desenvolvida para gerenciar vendas de agua de coco

## 🛠️ Tecnologias Utilizadas
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- Spring Security
- Jakarta EE
- Lombok
- Maven
- MySQL 8.0 / PostgreSQL
- JWT para autenticação
- Swagger/OpenAPI para documentação

## 💻 Pré-requisitos
- Java JDK 17+
- Maven 3.x
- MySQL 8.0 / PostgreSQL
- Git

## 🏃‍♂️ Como Executar

### 1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/parcao-back.git
cd parcao-back
```
